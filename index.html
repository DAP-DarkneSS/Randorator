================|Russian|=================

Рандоратор — это продвинутый графический генератор случайных чисел. Программа создаёт заданное количество значений в определённом интервале. Дополнительно можно указать среднее значение полученных чисел, максимальное относительное стандартное отклонение и количество знаков для округления. Присутствует возможность выбора разделителя в выводимых числах и автоматического копирования в буфер обмена.

================|English|=================

Randorator is an advanced graphical generator of random numbers. The program creates the specified number of values in a certain interval. Optionally, the average value of the received number, the maximum relative standard deviation and number of digits for rounding is able to specify. The possibility to automatically copy and select separator in the output numbers in the clipboard is present.
